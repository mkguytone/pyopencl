3.1. Pythonのインストール

Python公式サイトのパッケージをダウンロードするようにしてください。又、Mac OS Xについては、ActiveStateのインストールを先行してする必要があります。

以下が本書で使っている環境です。

  • Python 3.5.1
  • ActiveState 8.5.18

Mac OS XではデフォルトのPythonが存在し、これをアンインストールすることは推奨されていません。そのため、Python 3.5.1と、デフォルトのPythonが共存する形をとることになります。

以下が検証に使っている環境です。

$ which python
/usr/bin/python
$ python
Python 2.7.10 (default, Oct 23 2015, 19:19:21)
[GCC 4.2.1 Compatible Apple LLVM 7.0.0 (clang-700.0.59.5)] on darwin

$ which python3
/Library/Frameworks/Python.framework/Versions/3.5/bin/python3
$ python3
Python 3.5.1 (v3.5.1:37a07cee5969, Dec  5 2015, 21:12:44)
[GCC 4.2.1 (Apple Inc. build 5666) (dot 3)] on darwin

$ which pip3
/Library/Frameworks/Python.framework/Versions/3.5/bin/pip3
