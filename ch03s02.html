3.2. Pythonの基本パッケージのインストール

次にPyOpenCLで必要となる基本パッケージをインストールします。これに関してはパッケージ管理システムを使います。

$ pip3 install Mako
$ pip3 install numpy
$ pip3 install PyOpenGL
