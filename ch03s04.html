3.4. PIL（Python Image Library）のインストール

PILは画像pixelの読み込みに必要なライブラリですが、インストールはくせがあるため、PILがパッケージされているpillowをインストールすると良いでしょう。

$ pip3 install pillow
Collecting pillow
  Downloading Pillow-3.2.0-cp35-cp35m-macosx_10_6_intel.macosx_10_9_intel.macosx_10_9_x86_64.macosx_10_10_intel.macosx_10_10_x86_64.whl (3.0MB)
Installing collected packages: pillow
Successfully installed pillow-3.2.0

これだけでPILがインポートできるようになります。

注意点として、pillowはlibpngへの依存性があるため、pillowをインストールした時点でlibpngがインストールされてない場合は、PNGサポートなしでビルドされてしまいます。

これを避けるためにlibpngをあらかじめインストールしたほうがいいでしょう。

MacPortsの場合は以下のコマンドを、pillowの前にコマンドコンソールで叩いてください。

$ sudo port install libpng

HomeBrewの場合は以下のコマンドを使います。

$ sudo brew install libpng
